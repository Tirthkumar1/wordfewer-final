# Required Font Files

Place these font files in this directory before building:

## Plus Jakarta Sans (Headlines / Wordmark)
- PlusJakartaSans-Bold.ttf
- PlusJakartaSans-ExtraBold.ttf

## Nunito (Game word display)
- Nunito-Black.ttf
- Nunito-ExtraBold.ttf

## Inter (Body / UI labels)
- Inter-Regular.ttf
- Inter-Medium.ttf

## Noto Sans Indic Scripts
- NotoSansGujarati-Bold.ttf
- NotoSansDevanagari-Bold.ttf

All fonts are available from Google Fonts (fonts.google.com).
After placing the files here, register them in app.config.js under the expo-font plugin.
